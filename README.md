# Multiplayer Belote


## Installation

```
git clone git@github.com:FlorianDenis/Belote.git
cd Belote
virtualenv venv; source venv/bin/activate
pip install -r requirements.txt
```

## Launching the clent

```
python client.py belote.floriandenis.net
```
